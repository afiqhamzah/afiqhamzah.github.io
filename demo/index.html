Gridsome

Steps to get started


    • Install gridsome by running npm install -g gridsome
    • Then clone this repo by running git clone https://github.com/FriendlyUser/gridsome-starter-buefy.git
    • Update gridsome.config.js with unique site name and hosting details.
    • Change formspree to use your personal email address.
    Deployment can be handled using azure pipelines, additionally considering adding more customization.
